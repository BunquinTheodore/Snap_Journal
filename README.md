# 📸 Snap Journal

> *“Capture moments. Reflect on memories. Grow through journaling.”*

Snap Journal is a minimalistic yet powerful **Flutter** app designed to help users **capture, write, and reflect** on their daily thoughts.  
It combines simplicity, beauty, and emotional depth — perfect for students, writers, and anyone seeking a mindful digital journaling experience.

---

## 🌟 Features

✨ **Key Highlights**
- 📝 Create, edit, and delete journal entries  
- 🖼️ Attach photos from camera or gallery  
- 💾 Offline data storage using **Hive**  
- 🎨 Clean and Figma-accurate UI  
- 🔍 Search and browse through past entries  
- ⚙️ Responsive design for mobile and desktop  

---

## 🧠 Tech Stack

| Category | Technology |
|-----------|-------------|
| Framework | [Flutter](https://flutter.dev) |
| Language | Dart |
| Local Storage | [Hive](https://pub.dev/packages/hive) |
| State Management | Provider / SetState (depending on implementation) |
| UI Design | Custom Flutter widgets inspired by Figma design |
